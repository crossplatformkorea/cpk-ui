# Performance Snapshot

cpk-ui 0.7.0 removes the runtime IcoMoon selection file, adopts kstyled 0.4.1
strict compilation, and reduces the npm tarball from about 5.5 MB to 3.9 MB.

The release is covered by 21 Jest suites, 252 tests, public component and story
coverage checks, a static Storybook build, E2E package smoke tests, and an npm
tarball dry-run.

See the [performance guide](./PERFORMANCE.md) for architecture, current package
costs, validation commands, and consumer guidance.
