## Upgrade checklist

1. Upgrade `cpk-ui`, `kstyled`, and `babel-plugin-kstyled` together.
2. Enable `{strict: true}` for the kstyled Babel plugin.
3. Validate custom themes in light and dark modes.
4. Recheck controlled inputs, modal refs, and press handlers.
5. Run representative screens on iOS, Android, and web.
