## Install

```bash
bun add cpk-ui@0.7.0 kstyled@0.4.1
bun add --dev babel-plugin-kstyled@0.4.1
```

## Provider setup

```tsx
import {CpkProvider} from 'cpk-ui';

export function App() {
  return <CpkProvider>{/* application */}</CpkProvider>;
}
```

Use the theme toolbar to inspect light and dark behavior. Use the viewport
toolbar for compact mobile, mobile, tablet, and desktop layouts.
